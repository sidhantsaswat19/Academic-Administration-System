# Academic-Administration-System
Academic Administration System | **Java, OOP, Serialization**

**Architected a modular object-oriented system** to model **Many-to-One** relationships between Faculty and Students, utilizing **Abstract Classes** and **Polymorphism** to enforce data consistency.

**Implemented a custom persistence layer** using **Java Serialization** and **File I/O** to maintain the state of the UniversityRepository across application restarts.

**Utilized Java Collections (ArrayList)** to manage dynamic records of personnel, designing the system to scale for high-volume data processing.
